More Tip:

for cloud linux instance crontab to work, fix timezone
$ sudo timedatectl set-timezone <timeszone>
(equivalent cmd )
$ ln -s /usr/share/zoneinfo/Asia/Kolkata /etc/localtime

$ cat /etc/localtime

$ sudo service [cron|crond] restart
